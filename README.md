# 说明

- 帮助 [Docusaurus](https://docusaurus.io/zh-CN/docs)
- 项目 [pghpcdoc](https://gitee.com/jiyunqq/pghpcdoc.git)

### 依赖

- node.js >= 18.0
- yarn

### 包

```
中文文档搜索  https://github.com/easyops-cn/docusaurus-search-local
```

### 启动

```bash
yarn start       # 开发
yarn build       # 生成
yarn serve       # 启动 build 后的文档项目
```

git status
